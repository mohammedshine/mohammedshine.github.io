Security Advisory

Title: Stored and Reflected Cross-Site Scripting (XSS) Vulnerabilities in TeslaLogger Admin Panel

Advisory ID: TL-2024-001

Date: 18-May-2024

Severity: High

CVE ID: Pending Assignment

Affected Versions: TeslaLogger Admin Panel versions below 1.59.6

Patched Version: 1.59.6

Impact: Unauthorized actions, session hijacking, data theft

Attack Vector: Remote

Overview

Security researchers have identified multiple cross-site scripting (XSS) vulnerabilities in the TeslaLogger Admin Panel affecting versions below 1.59.6. These vulnerabilities allow attackers to inject malicious JavaScript payloads that can be stored or reflected in various web pages, leading to session hijacking, unauthorized actions, and data theft.

Vulnerabilities

1. Stored Cross-Site Scripting (XSS)

Affected URL: http://host:8888/admin/journeys.php

The "New Journey" field accepts unsanitized user inputs. Attackers can inject persistent JavaScript payloads, which are executed when the page is viewed by authenticated users.

Impact

Exploiting these vulnerabilities can allow attackers to:

  • Steal session cookies or credentials
  • Deface web pages
  • Execute arbitrary actions on behalf of authenticated users
  • Perform unauthorized data access

Remediation

It is highly recommended to update TeslaLogger Admin Panel to version 1.59.6 or later. Ensure proper input validation and sanitization, and apply appropriate encoding of user inputs before rendering them in HTML or JavaScript contexts.
